# ES6

### Proxy 代理

用于创建一个对象的代理，从而实现基本操作的拦截和自定义（如属性查找、赋值、枚举、函数调用等）

- get 方法：用于拦截某个属性的读取操作，可以接受三个参数，分别为目标对象、属性名和 proxy 实例本身
- set 方法：

```JS
let obj = {}
let proxy = new Proxy(obj,{
 get(){

 },
 set(target,key,value){
 }
})

```
